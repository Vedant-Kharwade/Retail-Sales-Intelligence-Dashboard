# 🏆 Retail Sales Intelligence Dashboard (Power BI Project)

An end-to-end data analytics project that analyzes **retail sales performance across products, cities, promotions, and time periods**, transforming raw transactional data into meaningful business insights.

Built using **Power BI and Excel**, this dashboard highlights trends in **sales, profit, quantity, product performance, and discount impact**.

## 📊 Key Insights

### 🛍️ Product Performance
- **Top-Selling Products (Revenue):**
  - Apple iPhone 14 (~21.4M)
  - Apple MacBook Air (~19.6M)
  - Sony Bravia TV (~19.4M)

- **Top Products by Profit:**
  - iPhone 14 (~2.14M)
  - MacBook Air (~1.96M)

- **Bottom Products:**
  - Colgate Toothpaste  
  - Dove Soap Pack  

👉 Indicates **low-margin FMCG items**

---

### 📦 Quantity Insights
- Highest quantity sold:
  - iPhone 14 (281 units)
  - Raymond Suit (274 units)

- Lowest quantity products:
  - Borosil Glass Set  
  - FabIndia Kurta  

👉 Shows **difference between premium & low-demand products**

---

### 💰 Sales vs Profit Relationship
- Strong **positive correlation**
- As sales increase → profit increases  

👉 Indicates **healthy pricing strategy**

---

### 🎯 Promotion Analysis
- **Summer Sale** → ~4.3M impact  
- **Weekend Flash Sale** → ~1.9M  
- **Clearance Sale** → ~1.0M  

👉 Discounts significantly impact **sales performance**

### 🌍 City-Level Insights
- Top performing cities:
  - Mumbai  
  - Bangalore  
  - Delhi  
  - Hyderabad  

👉 Metro cities drive majority of revenue

### 📈 Sales Trends

  - Promotional campaigns  

👉 Indicates **seasonality + marketing impact**

## 🔍 Workflow

1. **Data Collection**
   - Imported retail dataset (Orders, Products, Promotions)

2. **Data Cleaning**
   - Removed duplicates  
   - Fixed date formats  
   - Handled missing values  

3. **Data Transformation**
   - Created calculated fields:
     - Profit  
     - Discount %  
     - Net Sales  
     - Price per Unit  

4. **Data Modeling**
   - Built relationships between tables

5. **Visualization (Power BI)**
   - KPI Cards  
   - Bar Charts  
   - Map Visualization  
   - Scatter Plot  
   - Trend Analysis  

6. **Insight Generation**
   - Product performance analysis  
   - Regional insights  
   - Promotion effectiveness
